## Image Specifications

All images should be:
- **Format**: PNG
- **DPI**: 300 (for print quality)
- **Background**: White or transparent
- **Size**: Between 1200-2000 pixels wide

## Notes

1. Make sure all images exist before compiling the LaTeX document
2. If an image is missing, LaTeX will show an error
3. You can temporarily comment out missing images with `%` in the .tex file
4. All images are referenced with captions showing their source notebook

## Compiling the Presentation

Once all images are ready:
```bash
pdflatex presentation.tex
pdflatex presentation.tex  # Run twice for proper references
```
